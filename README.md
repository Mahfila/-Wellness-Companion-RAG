### Psychological Support FAQ Chatbot (Personal Project) | 2026
*Built a responsible AI wellness companion using RAG architecture*
- Developed a RAG-based chatbot using LangChain, ChromaDB, and Groq API (Llama 3) for mental health FAQ retrieval
- Implemented crisis-detection safety layer with keyword-based escalation to emergency resources (988 Lifeline)
- Designed ethical guardrails: clear AI disclaimers, boundary enforcement, and "I don't know" fallbacks
- Deployed interactive UI via Gradio with example prompts and mobile-responsive design
- Technologies: Python, LangChain, HuggingFace Embeddings, Chroma, Groq API, Gradio
- ⚠️ Disclaimer: Prototype for educational purposes only; not a clinical tool or replacement for professional care

### Technical Skills
- **AI/ML**: RAG architecture, LangChain, Prompt Engineering, HuggingFace Transformers
- **Backend**: Python, FastAPI (prototype), Vector Databases (Chroma, Pinecone)
- **Deployment**: Google Colab, Gradio, Groq API integration
- **Ethics & Safety**: Crisis detection protocols, AI boundary design, privacy-conscious development
